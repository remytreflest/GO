# Ex2 — Commandes

## Objectifs
- Compter des occurrences avec une `map[string]int`.
- Convertir une map en slice de structs pour pouvoir la trier.
- Trier par valeur décroissante avec `sort.Slice`.
- Filtrer et afficher uniquement les éléments apparaissant plus d'une fois.

---

Depuis la racine du module (`mira/`) :

```bash
go run ./go-warmup/ex2/ go api backend go rest go
```

Sans arguments (doit retourner exit code 1) :

```bash
go run ./go-warmup/ex2/
```

Build + exécution du binaire :

```bash
go build -o go-warmup/ex2/ex2.exe ./go-warmup/ex2/
./go-warmup/ex2/ex2.exe go api backend go rest go
```
